# 서비스 기능 설명
DESKIT
: 데스크테리어 상품을 소개하고 판매하는 라이브커머스 플랫폼입니다.
: 판매자는 라이브 방송을 진행하면서 상품을 판매할 수 있고, 회원들은 라이브 방송 시청을 통해 데스크테리어 상품에 대한 실사용감 등 체험적인 요소를 통해 상품 구매에 도움이 되는 정보를 얻을 수 있습니다. 

# 회원 대상 서비스
상품
: 데스크테리어 관련 상품을 둘러보고, 상세 정보(가격, 옵션, 이미지, 설명)를 확인하는 메뉴입니다.
: 태그(공간, 톤, 상황, 무드)를 필터로 검색할 수 있고, 원하는 상품 이미지 클릭시 상품 상세 메뉴로 이동됩니다.

셋업
: 실제 사용자들의 데스크 셋업 사례를 보고, 구성 아이템과 배치 아이디어를 참고하는 메뉴입니다.

라이브
: 실시간 방송에서 상품 소개를 보고, 혜택 정보를 확인하며 구매까지 할 수 있는 메뉴입니다. 
: 방송 시청과 동시에 실시간 채팅을 통한 판매자 및 회원들과의 양방향 소통이 가능합니다.

문의
: 상품, 주문, 배송, 교환/환불 등 궁금한 내용을 판매자 또는 고객센터에 문의하는 메뉴입니다.
: 필요시 관리자와 1:1로 궁금한 내용에 대한 상담을 진행할 수 있습니다.

장바구니
: 구매할 상품을 임시로 담아두고 수량/옵션을 정리한 뒤 한 번에 결제하는 메뉴입니다.

마이페이지
: 내 계정 정보, 주문내역, 배송지 관리, 정보 관리 등을 확인·관리하는 메뉴입니다.
: 내 MBTI와 직업 기반으로 데스크테리어 상품을 추천받을 수 있습니다(정보 수집 약관 동의시)

회원탈퇴
: 마이페이지에서 회원탈퇴를 진행할 수 있습니다.
: 탈퇴한 계정은 30일 이후 재가입 가능하며, 회원 정보는 관련 법령에 따라 일정 기간 보관되고 일부 데이터는 탈퇴시 복원 불가할 수 있습니다.

로그아웃
: 현재 로그인된 계정을 안전하게 종료하는 기능입니다.
