# DESKIT 기능 안내 (챗봇 참고용)

이 문서는 DESKIT 프로젝트의 기능을 챗봇이 질의 응답에 활용할 수 있도록 요약한 안내서입니다. 실제 구현은 백엔드(Spring)와 프론트(Vue) 코드 및 정책/프롬프트 문서에 기반합니다.

## 서비스 개요
- 라이브 커머스 + 데스크 셋업/상품 커머스를 결합한 플랫폼
- 사용자(구매자), 판매자, 관리자의 역할별 기능 제공
- 실시간 라이브 방송, VOD, 채팅, 결제, 주문, AI 챗봇/추천까지 포함

## 사용자(구매자) 기능
- 홈/카테고리 기반 상품·셋업 탐색, 인기 상품/셋업 노출
- 상품 상세, 셋업 상세, 태그 기반 탐색
- 라이브 방송 목록/상세 시청, VOD 시청
- 장바구니 담기/수정/삭제
- 주문 생성 및 결제(Toss), 결제 성공/실패 처리
- 주문 내역/상세 조회, 주문 취소·환불 요청
- 주소록 관리(추가/수정/삭제)
- 마이페이지/설정(프로필, 선호 정보 등) 관리
- 로그인/로그아웃, 토큰 재발급, 회원 탈퇴
- 고객 문의 챗봇 사용 및 상담 이관

## 판매자 기능
- 판매자 가입/등록, 상태·등급 관리
- 판매자 마이페이지 및 정보 관리
- 상품 등록/수정/상태 변경
- 상품 이미지 업로드, 태그 관리
- 주문 관리 및 상태 업데이트
- 라이브 방송 생성(기본 정보, 큐시트), 예약/진행
- 방송 상품 연동, 방송 통계 확인
- VOD 상세 조회 및 관리
- 판매자 탈퇴/정산 관련 처리

## 관리자 기능
- 관리자 인증 및 2차 인증(추가 검증)
- 사용자/판매자/회사 정보 관리
- 판매자 심사 및 평가 결과 관리
- 라이브 방송/예약/VOD 모니터링 및 관리
- 라이브 제재/규정 위반 관리 및 통계
- 고객지원/문의 관리
- 챗봇 대화/이관 내역 관리

## 커뮤니케이션/채팅 기능
- 라이브 채팅(WebSocket/STOMP, Redis Pub/Sub)
- 금칙어 필터링 및 제재 관리
- 1:1 문의(직접 채팅, 소켓 기반)
- 고객센터 탭/문의 UI

## 주문/결제 기능
- 주문 생성, 주문 상태 관리
- Toss 결제 승인/실패 처리
- 환불 및 취소 처리
- 배송비/할인 반영된 최종 결제 금액 계산

## AI/추천/챗봇 기능
- RAG 기반 챗봇 질의응답
- 문서 기반 라우팅 및 상담 이관
- 대화 저장/관리
- 사용자 선호 태그 기반 추천(Deskterior)
- 판매자 계획/평가 자동화 및 결과 알림

## 라이브/미디어 기능
- OpenVidu 기반 라이브 스트리밍
- 방송 예약/상태 관리
- 방송 상품 구성 및 레이아웃 관리
- VOD 생성/통계, 시청 기록 관리
- 이미지/미디어 업로드(S3 연동)

## 계정/보안/인증 기능
- 일반 로그인 및 JWT 기반 인증
- 소셜 로그인(카카오/네이버/구글)
- 휴대폰 인증(문자 발송/검증)
- 접근 권한 분리(사용자/판매자/관리자)

## 정책/지식 자산
- 주문/상품 정책 문서(`docs/policy/*`)
- 챗봇 프롬프트 템플릿(`prompt/*`)
- RAG 시드 문서(`src/main/resources/rag/seed/*`)
- 이메일 템플릿(`src/main/resources/email/*`)

## 참고 모듈(도메인)
- account: 회원/판매자/초대/주소/설정/인증
- product: 상품/이미지/태그
- setup: 셋업 및 태그 매핑
- order: 주문/결제/환불/판매자 주문
- cart: 장바구니
- livehost: 방송/예약/VOD/통계/제재
- livechat/directchat: 실시간 채팅/1:1 문의
- admin: 관리자 인증/관리/심사
- ai: 챗봇(RAG/OpenAI), 추천, 평가
